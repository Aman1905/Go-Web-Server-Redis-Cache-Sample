# Go Web Server with Caching using Redis
A sample Web Server program in Go that uses Redis to implement the Caching. This code repo is used in a blog post for DevRelSquad.

## Steps to run your program

**Step 1:** Install all the necessary dependencies stated in the article

**Step 2:** Follow the steps and code snippets as described in the article

**Step 3:** In the terminal, enter the following command to run your web server.

```bash
go run main.go
```

**Step 4:** Now, go to Postman and send a GET request to `http://localhost:8080/api/devrelData`.
